# Kellarakendus

## Autor
Kristo Kukk

## Kirjeldus
See on lihtne veebipõhine kellarakendus, mis näitab praegust aega, kuupäeva ja nädalapäeva.

## Funktsionaalsus
- **Aeg**: Näitab tunde, minuteid ja sekundeid.
- **Kuupäev**: Näitab päeva, kuud ja aastat.
- **Nädalapäev**: Näitab praegust nädalapäeva eesti keeles.
- **Tume režiim**: Saab vahetada heleda ja tumeda tausta vahel.
- **Kella formaat**: Saab valida 24-tunni või 12-tunni formaadi (AM/PM).
- **Sekundid**: Saab sisse ja välja lülitada sekundite näitamise.
- **Muusika**: Saab mängida taustamuusikat (sisse/välja).
- **Suurus**: Saab muuta kella teksti suurust + ja - nuppudega. Samuti saab kasutada klaviatuuri + ja - klahve.
- **Kujundus**: Kasutab gradient-tausta ja lihtsat, loetavat fonti (Courier New).

## Notes
Kasutasin AI väiksemate küsimuste jaoks (nt flexbox'i ja JS loopidega seotud küsimused).
Rakenduse kujundus sobitab end iga suurusega ekraanile (mobiil, tahvel, monitor jms).
