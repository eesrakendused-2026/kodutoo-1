# Funktsionaalsus
Suvaline taustavärv: muudab lehe taustavärvi suvaliseks värviks
Vaheta kellaaeg ja kuupäev: vahetab kella ja kuupäeva asukohti omavahel
Pluss/miinus: muudab fondi suurust (töötab ka klaviatuur + ja -)
Nooled: liigutab teksti paremale, vasakule, keskele
Vikerkaare kell: muudab kella teksti värviliseks
