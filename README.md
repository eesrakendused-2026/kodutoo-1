Vaade veebilehest
![veebilehe vaade](veebileht.png)

Veebilehel on erinevad funktsioonid:
Veebilehel saab muuta tausta pilti originaalvärvist ja tagasi
Veebilehel saab muuta kella fonti ja teksti värvi
Veebilehel saab vahetada AM/PM süsteemi ja 24h süsteemi vahel
Veebilehel saab muuta kella suurust
Veebilehel saab kutsuda esile jõevähi pildi plahvatustega
