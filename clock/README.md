Saskia Tork
[https://www.tlu.ee/~tork/2tund/clock.html](https://www.tlu.ee/~tork/2tund/clock.html)

Vastavalt kasutaja tegevusele on võimalik muuta järgmist: 
1. Vaate skaleerimine (Zoom In): Suurendab kella ja pleieri vaadet.
2. Vaate skaleerimine (Zoom Out): Vähendab kella ja pleieri vaadet.
3. Värviskeemi vahetus (Theme): Light Mode ja Dark Mode vahetus.
4. Disco: Vajutades discopallile hakkad pidu pihta. Taustavärvi muudatus ja muusika. 
5. Heli olek: Muusika esitamine ja peatamine (Play/Pause).Muusikat saab niisama ka käima panna ilma disco teemata. 
6. Helitugevus (Volume): Muusika valjuse reguleerimine liuguri abil.

 tehisintellekt aitas mind peamiselt vigade leidmisel ja JavaSkriptiga.
lisaks  tulid ka abimaterjalid kasuks.
